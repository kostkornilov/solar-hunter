# Назначение

Инструмент для оценки фильтров сглаживания цифровой модели рельефа (DEM). Сравнивает характеристики рельефа до и после применения различных фильтров сглаживания.

## Обзор

Этот инструмент:
1. Загружает ЦМР из области GeoJSON
2. Применяет несколько фильтров сглаживания (Гауссовский, Двусторонний, С сохранением признаков)
3. Вычисляет характеристики рельефа для исходных и сглаженных ЦМР
4. Генерирует статистический анализ и визуализации

## Установка

Требуется WhiteboxTools:
```bash
pip install whitebox
```


### С пользовательскими параметрами

```bash
python evaluate_smoothing_filters.py \
```

### Использование существующей ЦМР

```bash
python evaluate_smoothing_filters.py \
  --dem path/to/existing_dem.tif \
  --features slope,curvature \
  --filters bilateral
```

## Аргументы

- `--geojson PATH` (обязательно): Путь к файлу GeoJSON, определяющему интересующую область
- `--features FEAT1,FEAT2,...` (опционально): Список характеристик рельефа через запятую
  - По умолчанию: `reprojected_dem,slope,rugosity,terrain_ruggedness_index,aspect,roughness`
- `--filters FILT1,FILT2,...` (опционально): Список фильтров для тестирования через запятую
  - По умолчанию: `gaussian,bilateral,feature_preserving`
  - Доступные: `gaussian`, `bilateral`, `feature_preserving`
- `--output-dir PATH` (опционально): Выходная директория для результатов
  - По умолчанию: `filter_evaluation_output`

## Доступные характеристики
Нам нужно проверить 6 признаков. Для этого подходят **дефолтные значения параметров**.

Полный список (для сведения):
- `reprojected_dem` - Высота (Цифровая модель рельефа)
- `slope` - Уклон рельефа
- `aspect` - Экспозиция склона (направление уклона)
- `hillshade` - Отмывка рельефа
- `curvature` - Кривизна поверхности
- `planform_curvature` - Горизонтальная кривизна
- `profile_curvature` - Профильная кривизна
- `maximum_curvature` - Максимальная кривизна
- `topographic_position_index` - Топографический индекс положения (TPI)
- `terrain_ruggedness_index` - Индекс пересечённости рельефа (TRI)
- `roughness` - Шероховатость поверхности
- `rugosity` - Изрезанность поверхности

## Доступные фильтры

### Гауссовский фильтр
- **Скорость**: Очень быстрая (~0.1с)
- **Характеристики**: Простое сглаживание, не сохраняет границы
- **Лучше всего для**: Быстрое сглаживание, очень зашумлённые данные

### Двусторонний фильтр
- **Скорость**: Быстрая (~0.2с)
- **Характеристики**: С сохранением границ, поддерживает резкие границы
- **Лучше всего для**: Рельеф с резкими признаками (обрывы, хребты)

### С сохранением признаков
- **Скорость**: Средняя (~0.5с на итерацию)
- **Характеристики**: Адаптивное сглаживание с учётом рельефа
- **Лучше всего для**: Универсальное шумоподавление ЦМР (рекомендуется)

## Статистика (CSV)

Для каждого фильтра `statistics.csv` содержит:

| Столбец | Описание |
|---------|----------|
| `feature_name` | Название характеристики рельефа |
| `n_components_raw` | Количество связных областей в исходных данных |
| `n_components_smooth` | Количество связных областей после сглаживания |
| `n_comp_change_pct` | Процентное изменение количества компонентов |
| `morans_i_binary_raw` | Индекс Морана для исходной бинарной маски (пространственная автокорреляция) |
| `morans_i_binary_smooth` | Индекс Морана для сглаженной бинарной маски |
| `morans_i_change_pct` | Процентное изменение индекса Морана |
| `peak_retention_pct` | Процент диапазона значений, сохранённых после сглаживания |

**Интерпретация:**
- **n_components**(количество связных компонентов): Меньше значение → более однородные и непрерывные пространственные структуры
- **Moran's I**(индекс пространственной автокорреляции): Выше значение → сильнее пространственная кластеризация (соседние пиксели более похожи)
- **peak_retention**(сохранение экстремумов): Выше значение → лучше сохранены пиковые значения рельефа после фильтрации (100% = без потерь)

## Визуализации

### 1. Сравнение порогов
- Сетка 2×3, показывающая бинарные маски на квантилях 25%, 50%, 75%
- Верхний ряд: ИСХОДНЫЕ данные
- Нижний ряд: СГЛАЖЕННЫЕ данные
- Пороги квантилей вычисляются по ИСХОДНЫМ данным

### 2. Сравнение растров
- Непрерывные растры рядом
- Общая цветовая шкала (vmin/vmax)
- Показывает общий эффект сглаживания

### 3. Карта разностей
- Визуализация (Исходные - Сглаженные)
- Положительные значения (красный): Признаки, удалённые при сглаживании
- Отрицательные значения (синий): Признаки, добавленные/изменённые при сглаживании
- Показывает удаление шума против потери сигнала

## Примечания

- Пороги квантилей (25%, 50%, 75%) всегда вычисляются по ИСХОДНЫМ данным
- Значения NaN игнорируются во всех вычислениях
- Связные компоненты используют 8-связность (диагональные соседи включены)
- Индекс Морана использует упрощённое вычисление пространственной автокорреляции
- **Сохранение метаданных**: Все фильтры сглаживания сохраняют теги `original_bounds` из входных ЦМР
  - Это обеспечивает согласованное поведение обрезки для исходных и сглаженных расчётов рельефа
  - Позволяет точное создание карт различий и сравнение фильтров
